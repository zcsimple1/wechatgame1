# 幸运抽签小程序

一个简单易用的微信小程序，支持自定义选项和权重的抽签工具。

## 功能特点

- ✨ **自定义选项**: 用户可以自由添加任意数量的选项
- ⚖️ **权重设置**: 支持为每个选项设置权重，权重越大被抽中概率越高
- 🎲 **随机抽签**: 基于权重的随机算法，确保公平性
- 📊 **概率显示**: 实时计算并显示每个选项的抽中概率
- 🔄 **数据保存**: 自动保存选项到本地，下次打开自动恢复
- 🎉 **动画效果**: 抽签过程有动画效果和震动反馈
- 📤 **分享功能**: 生成分享页面，让好友也能参与抽签

## 使用说明

### 主页面功能

1. **添加选项**
   - 在输入框中输入选项名称
   - 可选设置权重（默认为1）
   - 点击"添加"按钮添加选项
   - 至少需要2个选项才能开始抽签

2. **管理选项**
   - 选项列表显示所有已添加的选项
   - 每个选项显示序号、名称、权重和概率
   - 点击右侧的 ✕ 按钮删除选项

3. **开始抽签**
   - 点击"开始抽签"按钮
   - 系统会快速随机切换选项（动画效果）
   - 最终根据权重随机选出一个选项
   - 抽签结果会以动画形式展示

4. **生成分享页**
   - 点击"生成分享页"按钮
   - 自动生成分享页面，包含所有选项信息
   - 可以将页面分享给好友

### 分享页面功能

- 显示所有选项列表
- 点击"开始抽签"即可参与抽签
- 点击"分享给好友"可以将抽签分享给更多人
- 支持通过链接分享，好友点击链接即可参与

## 权重说明

- 权重为可选字段，如果不设置默认为1
- 权重越大，该选项被抽中的概率越高
- 概率计算公式：某选项概率 = 该选项权重 / 所有选项权重总和 × 100%

例如：
- 选项A权重为1，选项B权重为2
- 选项A概率 = 1 / (1+2) × 100% ≈ 33.3%
- 选项B概率 = 2 / (1+2) × 100% ≈ 66.7%

## 开发说明

### 本地调试

1. 使用微信开发者工具打开项目
2. 选择项目目录
3. 填入AppID（测试可以使用测试号）
4. 点击"编译"即可预览

### 微信小程序相关配置

在微信小程序管理后台需要配置以下内容：

1. **服务器域名**（如需要后端接口）
2. **业务域名**（如需要webview）
3. **用户隐私保护指引**
